# easy-chat-pc

本项目为 easy-chat PC 端代码, 使用 react、typescript、vite 搭建。

* [PC端网站](https://easy-chat-xvos.onrender.com/)
* [后端项目地址](https://github.com/Loloao/easy-chat)

## Setup

* npm i

* npm run start

* 访问 localhost:5173
